# Política de publicación segura

## Sí se publica

- Técnicas, marcos, playbooks y aprendizajes generales
- Ideas, estrategias y plantillas reutilizables
- Casos abstractos sin datos personales

## No se publica

- Nombre real, teléfono, correo personal
- Dirección, ubicación exacta, rutina horaria exacta
- Universidad/campus específico, identificación institucional
- Capturas con datos privados o metadata sensible
- Tokens, llaves, secretos, IDs internos

## Regla de oro

Si una pieza permite inferir identidad real o ubicación precisa, no se publica.
