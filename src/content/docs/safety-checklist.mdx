# Checklist anti-doxx

Antes de publicar, confirmar:

- [ ] No hay nombre real ni contacto personal
- [ ] No hay dirección, ciudad precisa ni lugares frecuentes
- [ ] No hay horarios/rutinas exactas sensibles
- [ ] No hay datos de universidad/empresa que identifiquen persona
- [ ] No hay secretos, tokens, llaves o IDs internos
- [ ] Imágenes sin metadata sensible (EXIF)
- [ ] Texto revisado para evitar correlación de identidad
